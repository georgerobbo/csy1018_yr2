My Reflection

My time with CSY1018

Redoing web development has meant that I haven’t learnt HTML and CSS. I feel that this assignment has turned out much better than last year as it has meant that I could read the feedback I was given and try and work upon it as it is essentially a very similar assignment. One thing that has become apparent to me while doing this is that I really enjoy web development more than I thought giving me a good idea of what I am wanting to progress and do well in after more practice and learning more languages such as js and PHP in year 2.

The way the assignment is now done is something that I really enjoyed about this project as the self-research allows you to brush up on the module but also allows you to consider things that maybe you didn’t know or allowed you to research topics that interested you more.

The change from Atom being the suggested editor to VSC is a good one. Once set up, VSC is a great editor, especially the way you can integrate github into it. It made this module a breeze and I think it really shows in the amount of github commits as it saved all the hassle of using the git CMD.

I had a lot of fun with the module and feel like my skills from last year have progressed and hopefully that is validated through the grade I achieve.

It's validated!
